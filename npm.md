# NPM


### Save stuff with tilde by default
Run
```bash
npm config set save-prefix='~'
```
And from then on, you got it.

### Update dependencies globally
Run 
```bash
npm update -g
```

### Run multiple npm scripts at once
Run
```bash
npm-run-all <script-1> <script-2> ... <script-n>
```
